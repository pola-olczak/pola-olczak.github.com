## Cheat sheet for local running and deploy

Get the node version manager

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash
```

Get the latest node lts verion

```
nvm list --lts
```

Install that version:

```
nvm install vX.X.X
```


### Run it

```
npm start
```


### Deploy it

```
npm run deploy
```
